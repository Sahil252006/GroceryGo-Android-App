# GroceryGo - Mini Grocery Delivery App

A simple Blinkit-style grocery delivery Android application built using Kotlin and XML layouts.

## Features

* Login with OTP validation
* Product listing using RecyclerView
* Real-time product search
* Add to Cart functionality
* Quantity increase/decrease
* Remove cart items
* Checkout screen
* Payment method selection
* Order success screen
* Room Database integration
* Dark mode UI

## Tech Stack

* Kotlin
* XML Layouts
* RecyclerView
* ViewBinding
* Room Database
* Android Studio

## App Flow

Login → Home → Cart → Checkout → Success Screen

## How To Run

1. Clone repository
2. Open project in Android Studio
3. Sync Gradle
4. Run app on emulator or device
